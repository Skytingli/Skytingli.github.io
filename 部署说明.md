## GitHub Pages 部署

1. 登录 GitHub，新建仓库 `Skytingli.github.io`；如果仓库已存在，直接进入该仓库。
2. 解压本项目，将项目文件夹内部的全部文件上传至仓库根目录。不要把最外层文件夹作为额外一层上传。
3. 提交到默认分支 `main`。
4. 打开仓库 **Settings → Pages**。
5. 在 **Build and deployment** 中选择 **Deploy from a branch**，分支选择 `main`，目录选择 `/(root)`，保存。
6. 等待 GitHub Actions 构建完成，然后访问 `https://Skytingli.github.io`。
