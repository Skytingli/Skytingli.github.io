# 个人简介

<p class="lead">李婷，江西财经大学数字经济学院产业经济学博士研究生，主要研究方向为数字经济、产业经济、资源与环境经济学及金融经济学。</p>

目前主要关注人工智能与企业转型、网络媒体监督与企业绿色转型、数字技术与产业升级、供应链现代化以及金融司法改革等问题。研究注重结合中国经济实践，综合运用文本分析、机器学习与计量经济学方法开展实证分析。

# 研究方向

- **数字经济：**人工智能、数字化转型、金融科技与数字治理。
- **资源与环境经济学：**企业绿色转型、环境规制、网络媒体监督与绿色创新。
- **产业经济学：**新质生产力、供应链现代化与核心技术突破。
- **金融经济学：**金融司法改革、系统性金融风险与普惠金融。
