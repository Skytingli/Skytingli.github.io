# 李婷 Ting Li — Academic Homepage

这是基于 AcademicPages 模板精简后的中文单页学术主页，可直接部署至 `Skytingli.github.io`。

## 部署

将本项目全部文件上传到 GitHub 仓库 `Skytingli.github.io` 的默认分支，然后在仓库 **Settings → Pages** 中选择从该分支部署。页面地址为 <https://Skytingli.github.io>。

## 本地预览

```bash
bundle install
bundle exec jekyll serve
```

访问 <http://127.0.0.1:4000>。
