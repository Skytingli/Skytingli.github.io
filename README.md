# 李婷的中文学术主页

本项目是部署于 GitHub Pages 的中文学术个人主页，访问地址为：

<https://Skytingli.github.io>

## 部署方法

1. 登录 GitHub，进入名为 `Skytingli.github.io` 的仓库。
2. 将本项目中的全部文件和文件夹上传至仓库根目录。
3. 打开仓库的“Settings（设置）”→“Pages”。
4. 在“Build and deployment”中选择“Deploy from a branch”。
5. 分支选择 `main`，目录选择 `/(root)`，点击“Save”。
6. 等待一至数分钟后访问 <https://Skytingli.github.io>。

更新个人资料时，可直接修改对应的 Markdown 文件；替换头像时保留文件名 `assets/images/profile.jpg`，替换简历时保留文件名 `assets/files/LiTing_CV.pdf`。
